# Angular-Custom-UI (Project Library)
- Building Custom Form Control from scratch using ControlValueAccessorDirective
- Includes custom select, custom multi-select, custom checkbox, custom input
- All controls use shared validation component

# Angular WebApp (App)
- Small app spin-up to consume the library

# How to use
- use `npm link` on angular custom ui directory
- use `npm link custom-form-controls` on angular web app
- use `npm serve` on angular web app
